# ​​iOS 兼容性问题​

# ​问题​​：

iOS 可能因安全策略阻止 iframe 加载非 HTTPS 内容或限制媒体播放。 ​### ​解决方案​​： 在 pages.json 中配置允许任意加载：

{
  "path": "pages/iframe",
  "style": {
    "app-plus": {
      "allowsInlineMediaPlayback": "YES" // 允许内联媒体播放
    }
  }
}
